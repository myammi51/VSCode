# position property

  • position 속성을 설정한 요소는 다른 요소들을 무시하면서 마음대로 배치할 수 있게 된다 (3차원적인 요소가 된다)
  • static: 포지션을 설정하지 않은 상태
  • absolute: 절대적인 방식으로 위치를 설정
  • relative: 상대적인 방식으로 위치를 설정
  • fixed: 화면의 특정 부분에 고정될 위치를 설정
  • sticky: 원래 자리에 있다가 최종적으로 고정될 위치를 설정
  • 포지션 속성을 설정한 후에 top, bottom, left, right 속성을 통해 해당 요소의 위치를 결정할 수 있다

# position: absolute

  • absolute가 설정된 해당 요소의 부모 요소를 기준으로 절대적인 위치를 설정한다
  • 모든 부모 요소에 position이 설정되어 있지 않다면(static이라면) 화면을 기준으로 절대적인 위치를 설정한다

# position: relative

  • 해당 요소가 원래 있어야 하는 위치로부터의 위치를 설정한다
  • position: absolute는 본인이 원래 차지하고 있던 영역을 잃어버리지만 position: relative는 본인의 영역을 유지한다

# position: fixed

  • 화면에서 항상 고정된 위치를 차지하는 요소가 된다

# position: sticky

  • relative인척 하다가 fixed로 동작한다
