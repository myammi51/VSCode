# clear 속성 사용하기

  • clear: left - 해당 요소의 왼쪽에 float 요소가 올 수 없게 만든다
  • clear: right - 해당 요소의 오른쪽에 float 요소가 올 수 없게 만든다
  • clear: both - 해당 요소의 양쪽에 float 요소가 올 수 없게 만든다
# 컨텐츠의 양이 충분하지 않은 경우 float 요소가 박스를 벗어나게 된다 (해당 방향 clear를 통해 해결할 수 있다)
